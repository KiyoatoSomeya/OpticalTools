# OpticalTools
光学系ツール群（主に複数端末での共有用）

## 使用環境
-python 3.6.7
-openCV 3.4.5

## Capture Graycode
PySpinを利用してカメラでグレーコードパターンを投影およびカメラによる画像取得を行う。ARマーカーによる位置の取得を同時に行うことも可能。デフォルトの画像サイズは1920 * 1080。<br>

capture.pyによりカメラキャプチャを行う。
- 引数なしの場合marker.pngは利用しない
- 引数に"view"を指定することによって０番目のカメラでマーカーの位置を取得
- 引数に"world"を指定することによって１番目のカメラでマーカーの位置を取得
